# Antigravity Skills & Memory Repository

This repository acts as the continuous learning "brain" for the Antigravity AI assistant. It safely tracks learned workflows, scripts, and explicit behavioral corrections across all local projects.

## Architecture
- `bin/`: Executable Bash/Python skill scripts (must be prefixed with `ag-`).
- `summaries/`: Anonymized session summaries generated for cross-project pattern matching.
- `templates/`: Boilerplate configuration files for project scaffolding.
- `patches/`: Safe, system-generated unified diff patches for applying global configuration updates.
- `memories.md`: The core rule ledger for explicit human behavioral corrections.
- `INDEX.md`: A lightweight mapping index of all available skills.

**Note**: Do not commit raw secrets or `.env` files to this repository.
